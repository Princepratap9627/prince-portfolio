# Prince Pratap Singh - Premium Portfolio Website

A world-class, minimal, and futuristic single-page portfolio website designed to showcase projects and credentials in Artificial Intelligence, Machine Learning, Cybersecurity, and Web Engineering. 

Inspired by the design aesthetics of **Apple**, **Linear**, **Arc Browser**, and **Nothing**, it integrates a dark cyber theme with high-performance mathematical canvas interactions and fluid spring animations.

---

## ✨ Key Features

1.  **WWDC-style Mesh Background**: Slow-drifting purple (`#7C3AED`) and cyan (`#06B6D4`) gradient blobs rendered on an abstract grid, coupled with a mouse-following radial spotlight glow.
2.  **Cinematic Loading Reveal**: An elegant 1.5-second intro splash with glowing typography and progress animations, showing name, focus areas, and status.
3.  **3D Liquid Glass Orb**: A math-driven interactive 3D bubble rendered on HTML5 Canvas using sine-wave displacement, specular reflections, glass refraction layers, and mouse-displacement physics.
4.  **Raycast-style Floating Header**: A glassmorphic navigation dock that automatically transitions from transparent to blurred glass on viewport scroll. Includes a light/dark mode theme toggle.
5.  **3D Tilt Project Cards**: Linear-styled project grid showing detailed summaries for:
    *   **Employee Attrition Prediction System** (ML Risk Analyzer with interactive visual layouts)
    *   **RAKSHAK** (Enterprise Endpoint Detection & Response platform SOC telemetry visual)
    *   **PrepWise AI** (AI Interview Coach prompt dialog panel)
6.  **Visual Resume Preview**: An interactive, high-fidelity mock PDF reader rendering B.Tech credentials, skills, and projects, coupled with tab-open and direct download controls.
7.  **Apple Photos Certification Grid**: Clicking on any credential card animates it into a fullscreen verified badge modal utilizing Framer Motion's shared layout layout properties (`layoutId`).
8.  **Premium Contact Console**: Built-in clipboard copying utility with animation states alongside spring-scaling social links.

---

## 🛠️ Technology Stack

*   **Framework**: [React 19](https://react.dev/) + [Vite 6](https://vite.dev/) (Fast, lightweight ES modules)
*   **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (CSS-first engine with optimized compile speeds)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/) (Physics-driven animations and shared layout keys)
*   **Icons**: [Lucide React](https://lucide.dev/) (Clean outline icons)
*   **Visuals**: HTML5 Canvas 2D Context (No heavy WebGL wrappers, ensuring instant load times)

## 🚀 Getting Started

### Prerequisites
Make sure you have Node.js and npm installed. (The project has been compiled and tested successfully on Node `v24.14` and npm `v10.8.1`).

### 1. Installation
Clone the repository and install the dependencies:
```bash
# Navigate to the folder
cd prince-portfolio

# Install packages
npm install
```

### 2. Run Locally (Development Server)
Launch the local development server:
```bash
npm run dev
```
Open [http://localhost:5173](http://localhost:5173) in your browser to view the portfolio.

### 3. Build for Production
To compile and optimize the site for web hosting:
```bash
npm run build
```
This generates a production-ready bundle inside the `dist/` directory.

## 🌐 Deployment Guidelines

The project can be deployed instantly to modern cloud hosting platforms:

### Deploying to Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project root folder.
3. Keep the default settings (Vercel automatically detects Vite and configures the build command `npm run build` and output folder `dist`).

### Deploying to Netlify
1. Log in to Netlify and click **Add New Site** > **Import an existing project**.
2. Link your GitHub repository.
3. Configure build parameters:
   * **Build command**: `npm run build`
   * **Publish directory**: `dist`
4. Click **Deploy Site**.
